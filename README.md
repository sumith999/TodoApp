#  To-Do List App

This project is a functional To-Do List application for Android.It allows users to manage their daily tasks efficiently.

## üåü Features

*   **Add Tasks**: Easily add new tasks with a title, description, priority, and due date.
*   **Update Tasks**: Select and modify the details of any existing task.
*   **Delete Tasks**: Remove tasks you no longer need by long-pressing on them.
*   **Mark as Complete**: Use the checkbox to mark tasks as finished.
*   **Persistent Storage**: Your task list is saved to the device'''s local storage, ensuring your data is safe even after closing the app.
*   **User-Friendly Interface**: A clean and intuitive design for a seamless user experience.

## üöÄ How to Set Up and Run the Project

To run this project on your machine, please follow these steps.

### 1. Prerequisites

Make sure you have the latest version of **Android Studio** installed. You can download it from the [Android Developer website](https://developer.android.com/studio).

### 2. Clone the Repository

Open your terminal or command prompt and run the following command to clone the project:
```sh
git clone https://github.com/sumith999/Codsoft.git
```

### 3. Open in Android Studio

1.  Launch Android Studio.
2.  Select **File > Open** (or click **Open** on the welcome screen).
3.  Navigate to the cloned `TodoApp` directory and select it.
4.  Android Studio will recognize the project and start syncing it with Gradle, which downloads all the necessary dependencies.

### 4. Run the App

1.  Once the Gradle sync is complete, select a target device from the dropdown menu (either a physical device or an emulator).
2.  Click the **Run '''app'''** button (the green triangle icon ‚ñ∂Ô∏è).

Android Studio will now build, install, and launch the To-Do List app on your selected device.

## üì± How to Use the App

*   **To Add a task**: Fill in the title, description, and due date, then select a priority and click **Add Task**.
*   **To Update a task**: Simply tap on any task in the list. Its details will appear at the top. Make your changes and click **Update Task**.
*   **To Delete a task**: **Long-press** on the task you wish to remove.
*   **To Complete a task**: Tap the **checkbox** next to the task'''s title.
